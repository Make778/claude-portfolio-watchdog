## Введение

Claude Portfolio Watchdog — это персональный AI-наблюдатель за твоим
портфелем в Interactive Brokers. После 20-минутной настройки он:

- круглосуточно следит за 7 типами сигналов (инсайдеры, аналитики, цена, новости, earnings, макро, волатильность);
- присылает алерты в Telegram только когда событие действительно важно;
- каждое утро шлёт сводку, каждое воскресенье — премиум PDF-отчёт.

Никакого сервера, никакого Docker, никакой подписки. Всё работает на
бесплатном GitHub Actions.

## Шаг 1. Создание Telegram-бота

Telegram-бот — это «канал», через который система будет присылать тебе
сигналы и отчёты. Заведём его за 2 минуты.

1. Открой Telegram → в поиске введи **@BotFather** (это официальный бот
   Telegram, у него галочка верификации) → открой чат с ним.
2. Отправь команду `/newbot`.
3. BotFather спросит **имя бота** — введи любое, например
   `My Portfolio Watcher`. Это просто отображаемое имя в чате.
4. Дальше спросит **username** — он должен:
   - быть уникальным (если занят — попробуй другой),
   - оканчиваться на `bot`,
   - содержать только латиницу, цифры и подчёркивания.
   Пример: `temir_portfolio_bot`.
5. BotFather пришлёт сообщение с токеном — длинная строка вида
   `1234567890:ABCdefGhIJklMNOpqRsTUvwxyz_1234567890`. Это и есть
   **TELEGRAM_BOT_TOKEN**. Скопируй его в надёжное место (он понадобится
   на Шаге 6).
6. **🔴 ОБЯЗАТЕЛЬНО:** в сообщении BotFather будет ссылка на твоего бота
   (вида `t.me/temir_portfolio_bot`). Открой её → нажми кнопку **Start**.
   Без этого бот не сможет тебе писать (защита Telegram от спама).

## Шаг 2. Получение Chat ID

Chat ID — это твой уникальный «адрес» в Telegram (число вроде
`123456789`). Бот знает токен (право отправлять), но без Chat ID не
знает кому отправлять.

1. В Telegram найди бота **@userinfobot** (есть несколько с похожим
   именем — выбирай с большим числом запусков).
2. Открой с ним чат → нажми **Start**.
3. Он сразу пришлёт твой профиль в формате:
   ```
   Id: 123456789
   First: Temir
   Lang: ru
   ```
4. Число после `Id:` — это **TELEGRAM_CHAT_ID**. Скопируй его.

## Шаг 3. Получение Claude API-ключа

Claude — это AI, который «мозг» продукта. Он читает новости и пишет
аналитику. Платный, но недорогой (~$1.50 в месяц — см. [API_COSTS.md](docs/API_COSTS.md)).

1. Открой [console.anthropic.com](https://console.anthropic.com/) →
   **Sign Up** (или Login, если уже регистрировался).
2. Подтверди email + номер телефона. Российские номера принимаются.
3. Зайди в **Plans & Billing** → **Add credits** → пополни на **$5**
   (этого хватит на 3–4 месяца).
   - 💡 **Оплата для РФ:** российские карты не работают. Варианты:
     - Зарубежная карта (если есть)
     - Сервисы виртуальных карт (PayPal-карта, Wise, Revolut)
     - Попросить друга за рубежом — операция одноразовая
4. Перейди в **API Keys** в левом меню → **Create Key** → дай ключу любое имя
   (например `portfolio-watchdog`).
5. Скопируй полученный ключ (начинается с `sk-ant-api03-...`).
   **⚠️ ВНИМАНИЕ:** ключ показывают **один раз** — если закроешь окно,
   создашь новый. Сохрани в надёжное место — это и есть **CLAUDE_API_KEY**.

## Шаг 4. Форк репозитория

1. Открой репозиторий-шаблон в GitHub.
2. Нажми зелёную кнопку **Use this template** → **Create a new repository**.
3. Сделай репозиторий **Private**.
4. Дай ему любое имя (например, `my-portfolio-watchdog`).

## Шаг 5. Настройка `config.yaml`

Открой файл `config.yaml` через веб-редактор GitHub (карандашик).
Каждое поле подробно прокомментировано в [`config.example.yaml`](config.example.yaml).

Минимум, что нужно поменять:
- Список `tickers:` — твои бумаги (если ты не загрузишь CSV).
- `reports.language` и `reports.timezone` — под себя.
- `thresholds.price_movement_percent`, `thresholds.news_importance_min` —
  если хочешь меньше/больше шума.

Сохрани через **Commit changes**.

## Шаг 6. Добавление секретов

Settings → Secrets and variables → **Actions** → **New repository secret**.

Создай три секрета:

| Имя секрета | Что вставить |
|---|---|
| `TELEGRAM_BOT_TOKEN` | токен от @BotFather (Шаг 1) |
| `TELEGRAM_CHAT_ID` | ID из @userinfobot (Шаг 2) |
| `CLAUDE_API_KEY` | ключ от console.anthropic.com (Шаг 3) |

## Шаг 7. Загрузка портфеля из IBKR

1. Зайди в IBKR Client Portal → Performance & Reports → Statements.
2. Выбери период (например, последние 30 дней).
3. Format: **CSV**, скачай файл.
4. На GitHub открой папку `portfolio/` → **Add file** → **Upload files**.
5. Загрузи CSV. Commit changes.

Подробнее о поддерживаемых форматах — в [`portfolio/README.md`](portfolio/README.md).

**Не хочешь возиться с IBKR?** Просто перечисли тикеры в `config.yaml` —
система обойдётся без CSV.

## Шаг 8. Включение Actions и первый запуск

1. Открой вкладку **Actions** в репозитории.
2. Если видишь предупреждение про disabled workflows — нажми **I understand my workflows, go ahead and enable them**.
3. В левой колонке выбери `Portfolio Monitor` → **Run workflow** → Run.
4. Через минуту в Telegram должно прийти приветственное сообщение.
