# История изменений

## [1.0.0] — 2026-05-16

### Первый релиз

#### Возможности
- **7 типов мониторинга**: инсайдеры (SEC Form 4), рейтинги аналитиков,
  движения цены, новости (с фильтром Claude), earnings, макро, волатильность.
- **Ежедневная утренняя сводка** в Telegram.
- **Еженедельный PDF-отчёт** на 10+ страниц с графиками
  (тёмная и светлая темы, акцентный цвет настраивается).
- **Риск-аналитика**: Beta, корреляции, максимальная просадка, VaR, концентрация.
- **Сравнение с бенчмарком**, топ-лидеры и аутсайдеры, коэффициент Шарпа.
- **AI-наблюдения от Claude** для executive summary, разбора позиций,
  комментариев к рискам, прогнозам и рекомендациям.
- **Импорт CSV из IBKR**: Activity Statement / Flex Query / Portfolio Analyst /
  Simple — с автоматическим определением формата.
- **4 workflow в GitHub Actions**: hourly monitor, daily summary, weekly PDF,
  demo (запуск вручную).
- **Welcome-сообщение** при первом запуске.
- **Демо-режим** с примерами всех типов уведомлений.
- **Healthcheck-скрипт** (`python -m src.healthcheck`).

#### Технический стек
- Python 3.11+, pydantic v2 для валидации конфига.
- anthropic ≥0.39, tenacity для retry, yfinance для рыночных данных,
  reportlab + matplotlib для PDF.
- pytest-тесты с моками внешних API.
- Black + Ruff в pre-commit; detect-secrets для предотвращения утечек.

#### Что настраивается через config.yaml
- Список тикеров, пороги триггеров, расписание cron, язык интерфейса
  (ru / en), бенчмарк, таймзона, валюта, тема PDF, акцентный цвет,
  стиль наблюдений Claude.
